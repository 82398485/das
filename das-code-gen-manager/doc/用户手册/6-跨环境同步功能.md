&nbsp;&nbsp;&nbsp;&nbsp; 1、先同步组信息，如果组已经存在则无需同步

&nbsp;&nbsp;&nbsp;&nbsp; 2、然后同步项目所需的物理库，再更新物理库在当前环境的连接信息

&nbsp;&nbsp;&nbsp;&nbsp; 3、再同步逻辑库和逻辑库映射
